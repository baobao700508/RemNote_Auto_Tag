# 安装指南 - 目录知识结构排序插件
# Installation Guide - Directory Knowledge Structure Sorting Plugin

## 安装步骤
## Installation Steps

### 方法一：通过插件商店安装（推荐）
### Method 1: Install via Plugin Store (Recommended)

1. 打开RemNote应用
1. Open the RemNote application

2. 点击左下角的"设置"图标
2. Click the "Settings" icon in the lower left corner

3. 选择"插件"选项卡
3. Select the "Plugins" tab

4. 搜索"目录知识结构排序"或"Auto Tag"
4. Search for "Directory Knowledge Structure Sorting" or "Auto Tag"

5. 点击"安装"按钮
5. Click the "Install" button

### 方法二：手动安装
### Method 2: Manual Installation

1. 下载最新版本的`PluginZip.zip`文件
1. Download the latest version of the `PluginZip.zip` file

2. 打开RemNote应用
2. Open the RemNote application

3. 点击左下角的"设置"图标
3. Click the "Settings" icon in the lower left corner

4. 选择"插件"选项卡
4. Select the "Plugins" tab

5. 点击"从文件安装"
5. Click "Install from File"

6. 选择下载的`PluginZip.zip`文件
6. Select the downloaded `PluginZip.zip` file

7. 确认安装
7. Confirm installation

## 安装后的初始设置
## Initial Setup After Installation

1. 安装完成后，重启RemNote应用
1. After installation is complete, restart the RemNote application

2. 在左侧边栏找到"目录知识结构排序"图标并点击
2. Find the "Directory Knowledge Structure Sorting" icon in the left sidebar and click it

3. 使用开关按钮开启自动标记功能（默认为开启状态）
3. Use the toggle button to enable the automatic tagging function (enabled by default)

## 使用权限确认
## Permission Confirmation

首次使用时，RemNote可能会请求确认插件的权限要求：
When using for the first time, RemNote may request confirmation of the plugin's permission requirements:

- 当看到权限请求对话框时，请点击"允许"以授予插件所需的权限
- When you see the permissions request dialog, click "Allow" to grant the plugin the necessary permissions

- 这些权限仅用于在您的本地RemNote中读取和添加标签
- These permissions are only used to read and add tags in your local RemNote

## 常见问题解决
## Common Issues and Solutions

### 问题：插件安装后没有显示在侧边栏
### Issue: Plugin not showing in the sidebar after installation

- 解决方法：重启RemNote应用，或检查插件是否已正确安装和启用
- Solution: Restart the RemNote application, or check if the plugin is correctly installed and enabled

### 问题：标签没有自动添加
### Issue: Tags are not automatically added

- 解决方法：
- Solution:

  1. 确认自动标记功能已开启（侧边栏面板中的开关按钮为绿色）
  1. Confirm that the automatic tagging function is enabled (the toggle button in the sidebar panel is green)

  2. 检查是否已正确添加"Content Structure Sign"标签到Rem
  2. Check if the "Content Structure Sign" tag has been correctly added to the Rem

  3. 确认新创建的Rem的时间晚于添加标签的时间
  3. Confirm that the creation time of the new Rem is later than the time when the tag was added

### 问题：跳转按钮无法正常工作
### Issue: Jump buttons not working properly

- 解决方法：
- Solution:

  1. 确认您有访问被标记Rem的权限
  1. Confirm you have permission to access the tagged Rem

  2. 检查Rem是否仍然存在（未被删除）
  2. Check if the Rem still exists (has not been deleted)

  3. 尝试刷新应用后再次尝试
  3. Try refreshing the application and try again

2. 点击左下角的"设置"图标
2. Click the "Settings" icon in the lower left corner

3. 选择"插件"选项卡
